# StarryLobby

## 介绍
一款 Minecraft 大厅插件，基于 Spigot 1.12.2 实现。

## 依赖
- **[必须]** 插件本体基于 Java 11, Spigot API 1.12.2 实现。
- **[扩展]** 变量获取基于 PlaceHolderAPI 实现。
- **[扩展]** 称号获取基于 LuckPerms 实现。

## 命令

<details>
    <summary>可用命令</summary>

```text
# /edit 
- 开放编辑地图。

# /crash <玩家>
- 崩溃玩家客户端
```
</details>

## 准备制作的功能
- PlayerSettings集成及多功能
- 大厅等级系统
- 匹配系统
- 数据库储存（MySQL Database）
